# RetroCore Windows Simulator 🌐🎮

Welcome to **RetroCore**, a high-fidelity retro Windows operating system simulation built entirely using HTML, CSS, and vanilla JavaScript. This project is lightweight, fully interactive, and designed to run natively in any modern web browser.

## 🚀 Live Demo
Once GitHub Pages is activated, you can access the simulator live here:
`https://github.com/anyelo888ra-ux/retrocore-windows-sim`

## 🌟 Key Features

* **Classic Windows UI:** Features a fully movable window system, functional close/minimize systems, and a real-time taskbar clock.
* **Working Start Menu:** Access core features, reload the operating system environment, or launch applications directly.
* **Built-in Games:** Includes a retro 2D **Snake Clásico** arcade game with real-time score tracking built inside an HTML5 canvas.
* **Interactive 2-Stage Virus Simulator:** A harmless cyber-security experiment to show how early staged-malware behaved:
  * **Stage 1:** Causes window border destabilization and registry alerts.
  * **Stage 2:** Restricts mouse layout patterns, alters core styling, and triggers a full-scale classic **Blue Screen of Death (BSOD)** with realistic technical audio alerts.

## 🛠️ Installation & Setup

1. Create a new repository on GitHub named `retrocore-windows-sim`.
2. Upload the `index.html` file to the main folder.
3. Go to **Settings -> Pages** inside your repository.
4. Set the Build branch to `main` (or `master`) and hit **Save**.
5. Refresh after 2 minutes to access your simulation link!
